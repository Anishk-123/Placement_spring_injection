# Placement Spring Injection Project

## Overview
This is a Java project demonstrating Spring Framework dependency injection.

## Structure
- `src/com/beans`: Model classes (Student, Address).
- `src/com/main`: Main application class.
- `src/com/resources/applicationContext.xml`: Spring configuration.

## How to Run
1. Ensure JDK and Maven/Gradle setup.
2. Run `Main.java`.
